# Notes App

## 📌 Description
A simple and interactive Notes App built using HTML, CSS, and JavaScript. Users can create, edit, and delete notes easily.

## 🚀 Features
- Add new notes
- Edit existing notes
- Delete notes
- Data stored using localStorage

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 💡 What I Learned
- DOM manipulation
- Handling events in JavaScript
- Using localStorage for data persistence
